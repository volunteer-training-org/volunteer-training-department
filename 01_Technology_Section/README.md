# قسم التقنية

## نبذة
يعنى قسم التقنية بدعم التحول الرقمي لإدارة التدريب التطوعي،
من خلال تطوير الأنظمة والأدوات التقنية،
وتحليل البيانات، وبناء لوحات متابعة تدعم اتخاذ القرار.

## نطاق العمل
يشمل عمل قسم التقنية ما يلي:
- تطوير برنامج أو نظام خاص بإدارة التدريب التطوعي.
- تحليل احتياجات المستخدمين (الإدارة والمتدربين).
- إعداد لوحات البيانات والتقارير التقنية.
- دعم تكامل البيانات مع الأنظمة والأدوات المستخدمة.
- تحسين جودة البيانات وضمان سلامتها.

## المشاريع الحالية
- تطوير نظام إدارة التدريب التطوعي.
- توثيق المتطلبات التقنية والوظيفية للنظام.
